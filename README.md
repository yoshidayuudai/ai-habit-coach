# AI習慣コーチ

##  概要
習慣を記録し、AIから改善アドバイスを受けられるWebアプリです。  
日々の行動を可視化し、継続しやすい環境を作ることを目的としています。

---

##  デモ
（ここに後でURL貼る）

---

##  主な機能

- 習慣の追加
- 習慣の記録（ワンクリック）
- AIによる習慣分析・改善提案

---

##  AI機能の特徴

AIが以下の3点を自動生成します：

- 良い点
- 改善点
- 明日やるべき行動

 単なるアドバイスではなく「行動に落とし込める設計」にしてる

---

## 🛠 使用技術

- Next.js（App Router）
- TypeScript
- Tailwind CSS
- OpenAI API

## 工夫した点

- APIエラー時のハンドリング
